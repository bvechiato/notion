Inheritance

Child classes inherit everything from parent class


Polymorphism

Overriding the implementation of a function in a child class

Reminder: poly (many) morphism (shaped)!


Encapsulation

Visibility of a variable outside class definition: public (visible outside), private (invisible outside)


Abstraction

An abstract class is general, acts like a rule book for child classes

1) Child classes must implement all abstract functions

2) Abstract classes cannot be instantiated/have objects

from abc import ABC, abstractmethod
